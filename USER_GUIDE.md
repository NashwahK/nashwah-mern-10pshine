## Getting Started

### Sign Up

1. Open Pendown in your browser
2. Click the **Sign Up** link on the login page
3. Enter:
   - **Username** (3+ characters, letters/numbers/underscores only)
   - **Email** (valid email address)
   - **Password** (6+ characters)
   - **First Name** (optional)
   - **Last Name** (optional)
4. Click **Sign Up**
5. You'll be redirected to login. Enter your credentials and click **Login**

### Log In

1. Enter your email address
2. Enter your password
3. Click **Login**
4. You'll see the main notes view

### Reset Your Password

If you forgot your password:

1. On the login page, click **Forgot password?**
2. Enter your email address
3. Check your email for a reset link (valid for 30 minutes)
4. Click the link and enter your new password
5. Return to login and use your new password

### Log Out

Click the **Logout** button in the sidebar under your email. You'll be returned to the login page.

**Note:** If you close the browser without logging out, you'll be automatically logged out for security.

## Creating a Note

### Method 1: Create New Note

1. Click the **New Note** button in the left sidebar
2. Enter a title (required)
3. Click in the content area and type or paste your content
4. The note saves automatically as you type
5. You'll see a success notification when saved

### Method 2: From Scratch

1. Navigate to the Notes view
2. Click **+ New Note** button (top-right area)
3. Start typing your note

### Rich Text Formatting

Pendown includes a powerful text editor with formatting options:

- **Bold** - Select text, click the Bold icon (or Ctrl+B)
- **Italic** - Select text, click the Italic icon (or Ctrl+I)
- **Underline** - Select text, click the Underline icon (or Ctrl+U)
- **Strikethrough** - Select text, click the Strikethrough icon
- **Code Block** - Click the Code icon or use Ctrl+Shift+C
- **Lists** - Click the Bullet or Numbered list icon
- **Indent/Outdent** - Use Tab/Shift+Tab in lists
- **Headings** - Use the formatting dropdown (H1, H2, H3, etc.)
- **Quotes** - Click the Quote icon to create block quotes
- **Links** - Select text, click Link icon, enter URL
- **Clear Formatting** - Select text, click Clear Formatting

### Adding Tags

While editing a note:

1. Look for the tag input field
2. Type a tag name and press Enter
3. Tags help organize and filter your notes
4. Multiple tags per note are supported

### Setting a Note Color

1. Open a note for editing
2. Click the color picker icon
3. Select a color from the circular palette (7 colors available)
4. Or enter a custom hex color code (e.g., #FF5733)
5. Click outside to close the color picker
6. The note color is saved immediately

### Unsaved Changes Warning

If you've made changes to a note and try to leave:

1. A confirmation dialog appears: "Are you sure you want to leave this wisdom behind?"
2. Click **Leave** to discard changes
3. Click **Stay** to continue editing
4. Changes are never lost if you stay

## Editing a Note

1. Navigate to the Notes view
2. Click any note card to open it for editing
3. Modify the title, content, color, or tags
4. Changes save automatically
5. A success notification confirms each save

### Managing Note Title

1. Click the title field
2. Edit the text
3. Press Enter or click away to save

### Updating Content

1. Click in the content area
2. Make changes using the rich text editor
3. Changes save automatically every few seconds

### Changing Note Color

1. Click the color picker icon while editing
2. Select a new color
3. Click outside to apply
4. Note color updates immediately

## Deleting a Note

1. Open a note or view it in the list
2. Click the delete icon (trash can) on the note
3. A confirmation dialog appears: "Once you delete, there's no going back. Be certain."
4. Click **Delete** to confirm
5. The note is removed and you'll see a success notification

**Warning:** Deleted notes cannot be recovered. Be careful when deleting.

## Organizing Notes

### Using Tags

Tags are keywords that help organize and find notes quickly.

**Add a tag:**
1. While editing a note, look for the tag input area
2. Type a tag name (e.g., "work", "personal", "ideas")
3. Press Enter to add the tag
4. Multiple tags per note help with organization

**Remove a tag:**
1. While editing a note, find the tag you want to remove
2. Click the X icon next to the tag name
3. The tag is removed immediately

### Filter by Tag

In the Notes list view:

1. Look at the available tags displayed as chips/buttons
2. Click any tag to filter notes
3. Only notes with that tag appear
4. Click again to deselect the tag
5. Select multiple tags to see notes with any of those tags

## Searching and Filtering

### Search Notes

1. In the Notes view, locate the search bar at the top
2. Type keywords to search by title or content
3. Results update instantly as you type
4. Clear the search box to see all notes again

### Filter by Tags

1. Click on tag chips below the search bar
2. Single select shows notes with that tag
3. Multiple selections show notes with any selected tags
4. Click a selected tag again to deselect

### Sort Notes

1. Click the **Sort** dropdown in the Notes view
2. Choose from:
   - **Newest First** - Most recently created/edited
   - **Oldest First** - Least recently created/edited
   - **A → Z** - Alphabetical order
   - **Z → A** - Reverse alphabetical order
3. Notes reorder based on your selection

### Combine Search, Filter, and Sort

You can use all three together:

1. Search for "project" in the search bar
2. Filter by the "work" tag
3. Sort by "Newest First"
4. See only recent work notes containing "project"

## Tips and Tricks

### Keyboard Shortcuts

Use these shortcuts while editing:

- **Ctrl/Cmd + B** - Bold
- **Ctrl/Cmd + I** - Italic
- **Ctrl/Cmd + U** - Underline
- **Ctrl/Cmd + Z** - Undo
- **Ctrl/Cmd + Shift + Z** - Redo

### Quick Organization

1. Use consistent tag names for easy filtering
2. Create tags for projects, contexts, or categories
3. Color-code important notes for quick visual scanning

### Writing Tips

- Use headings (H1, H2, H3) to structure long notes
- Break content into sections with subheadings
- Use bullet points or numbered lists for clarity
- Quote important information with block quotes

### Link Notes

1. While editing, select text to make a link
2. Click the Link icon in the toolbar
3. Enter a URL (must start with http:// or https://)
4. Click or press Enter to create the link

### Clear Unwanted Formatting

1. Select the text with bad formatting
2. Click **Clear Formatting** icon in the toolbar
3. Text reverts to plain style

### Undo Changes

- While editing, press Ctrl+Z (or Cmd+Z on Mac) to undo
- Keep pressing to undo multiple actions
- Press Ctrl+Shift+Z to redo

## Troubleshooting

### My note isn't saving

**Solution:**
1. Check your internet connection
2. Ensure you're logged in (check sidebar for your email)
3. Look for error notifications at the bottom of the screen
4. Refresh the page and try again
5. Check browser console (F12) for error messages

### I can't log in

**Solution:**
1. Verify your email address is correct
2. Check that Caps Lock is off
3. Use "Forgot password?" to reset if needed
4. Try a different browser
5. Clear cookies and try again

### A note appears blank when I open it

**Solution:**
1. Refresh the page (F5 or Cmd+R)
2. Try logging out and back in
3. Check browser console for errors
4. Contact support if issue persists

### The editor toolbar is missing

**Solution:**
1. Refresh the page
2. Make sure the note is fully loaded
3. Check that JavaScript is enabled in your browser

### I accidentally deleted a note

**Solution:**
Unfortunately, deleted notes cannot be recovered. To prevent accidental deletion:
- Use the confirmation dialog carefully
- You'll be asked to confirm before deletion
- If you deleted recently, you may contact support

### Colors aren't applying to notes

**Solution:**
1. Make sure you've selected a color in the color picker
2. Click outside the color picker to save
3. Refresh the page to verify
4. Try a different color

### Search or filter isn't working

**Solution:**
1. Refresh the page
2. Check that you're using correct tag names (case-sensitive)
3. Clear search and try again
4. Try filtering by just one tag first

### I'm being logged out unexpectedly

**Solution:**
This happens when:
- You close the browser (intended security feature)
- Your session expires (7 days)
- You clear browser cookies manually
- You log out explicitly

To stay logged in, keep the browser window open.
